## 边界条件测试

### 权限验证测试

| 测试场景 | 操作 | 预期结果 | 状态 |
|----------|------|----------|------|
| 未登录访问供应商页面 | 直接访问 `/supplier/dashboard` | 跳转到供应商登录页 | |
| 用户访问供应商接口 | 用户登录后调用 `/api/supplier/orders` | 返回401或403 | |
| 管理员访问供应商接口 | 管理员登录后调用 `/api/supplier/orders` | 返回401（需要供应商token） | |
| 供应商操作他人订单 | 供应商A尝试操作供应商B的订单 | 返回"无权操作此订单" | |

### 数据验证测试

| 测试场景 | 操作 | 预期结果 | 状态 |
|----------|------|----------|------|
| 注册时用户名重复 | 使用已存在的用户名注册 | 返回"用户名已存在" | |
| 登录账号未激活 | 使用pending状态账号登录 | 返回"账号未激活或已被停用" | |
| 重复确认接单 | 对已确认的订单再次确认 | 返回"订单状态不允许此操作" | |
| 未确认直接发货 | 对pending状态订单发货 | 返回"订单状态不允许发货" | |
