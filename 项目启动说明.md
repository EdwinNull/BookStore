# 🚀 书店管理系统项目启动说明

## 📋 项目概述

书店管理系统是一个基于 Spring Boot + Vue 3 的全栈应用，包含：
- **后端**: Spring Boot 3.4.0 + MyBatis + MySQL
- **前端**: Vue 3 + TypeScript + Element Plus + Vite
- **数据库**: MySQL 8.0+
- **缓存**: Redis

## 🛠️ 环境要求

### 必需软件
- **JDK 17+** - Java开发环境
- **Maven 3.6+** - 项目构建工具
- **Node.js 16+** - 前端开发环境
- **MySQL 8.0+** - 数据库
- **Redis 6.0+** - 缓存服务（可选）

### 开发工具（推荐）
- **IDE**: IntelliJ IDEA / VS Code
- **数据库管理**: Navicat / DBeaver
- **API测试**: Postman / Apifox

## 🚀 完整启动流程

### 第一步：环境准备

1. **安装Java环境**
   ```bash
   # 检查Java版本
   java -version
   # 需要JDK 17+
   ```

2. **安装Maven**
   ```bash
   # 检查Maven版本
   mvn -version
   # 需要Maven 3.6+
   ```

3. **安装Node.js**
   ```bash
   # 检查Node.js版本
   node --version
   npm --version
   # 需要Node.js 16+
   ```

4. **启动MySQL服务**
   ```bash
   # Windows
   net start mysql

   # 或者通过服务管理器启动MySQL服务
   ```

5. **启动Redis服务（可选）**
   ```bash
   # Windows
   redis-server

   # 或者通过服务管理器启动Redis服务
   ```

### 第二步：数据库配置

1. **创建数据库**
   ```sql
   -- 连接到MySQL
   mysql -u root -p

   -- 执行数据库创建脚本（项目已包含）
   source d:/ProjectCollection/BookStore/database/bookstore_schema.sql
   ```

2. **验证数据库**
   ```sql
   -- 检查数据库是否创建成功
   SHOW DATABASES;
   USE bookstore;
   SHOW TABLES;
   ```

3. **检查初始数据**
   ```sql
   -- 查看默认管理员账户
   SELECT username, name, role FROM users WHERE username = 'admin';

   -- 预期结果：admin / 系统管理员 / admin
   ```

### 第三步：后端项目启动

1. **进入后端目录**
   ```bash
   cd d:/ProjectCollection/BookStore
   ```

2. **检查数据库配置**
   - 确认 `src/main/resources/application.yml` 配置正确：
     ```yaml
     spring:
       datasource:
         url: jdbc:mysql://localhost:3306/bookstore?...
         username: root
         password: 3107344108  # 你的MySQL密码
     ```

3. **编译并启动后端**
   ```bash
   # 使用Maven编译项目
   mvn clean compile

   # 启动Spring Boot应用
   mvn spring-boot:run
   ```

4. **验证后端启动成功**
   - 查看控制台输出，应该显示：
     ```
     Tomcat started on port 8080 (http)
     Started BookStoreApplication in 2.36 seconds
     ```
   - 访问：http://localhost:8080
   - 预期看到404错误（这是正常的，因为没有根路径）

5. **测试后端接口**
   ```bash
   # 测试用户登录接口
   curl -X POST http://localhost:8080/api/login \
     -H "Content-Type: application/x-www-form-urlencoded" \
     -d "username=admin&password=admin123"
   ```

### 第四步：前端项目启动

1. **打开新的命令行窗口**

2. **进入前端目录**
   ```bash
   cd d:/ProjectCollection/BookStore/frontend
   ```

3. **安装依赖**
   ```bash
   npm install
   ```

4. **启动前端开发服务器**
   ```bash
   npm run dev
   ```

5. **验证前端启动成功**
   - 查看控制台输出，应该显示：
     ```
     VITE v5.4.20  ready in 591 ms
     ➜  Local:   http://localhost:3001/
     ```

### 第五步：完整功能测试

1. **访问首页**
   - 打开浏览器访问：http://localhost:3001
   - 应该看到书店管理系统首页

2. **用户登录测试**
   - 访问：http://localhost:3001/login
   - 使用测试账号：
     - 用户名：admin
     - 密码：admin123

3. **管理员登录测试**
   - 访问：http://localhost:3001/admin/login
   - 使用管理员账号登录

4. **注册新用户**
   - 访问：http://localhost:3001/register
   - 注册新用户进行测试

## ⚙️ 环境配置说明

### 后端配置文件：`application.yml`

```yaml
spring:
  datasource:
    # 数据库连接配置
    url: jdbc:mysql://localhost:3306/bookstore?useUnicode=true&characterEncoding=utf8mb4&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: 你的MySQL密码
    driver-class-name: com.mysql.cj.jdbc.Driver
    # 连接池配置
    hikari:
      maximum-pool-size: 10
      minimum-idle: 5
      connection-timeout: 20000

  # Redis配置（可选）
  data:
    redis:
      host: localhost
      port: 6379
      timeout: 2000ms

# MyBatis配置
mybatis:
  configuration:
    map-underscore-to-camel-case: true
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
  mapper-locations: classpath:com/weblearning/bookstore/mapper/*.xml
```

### 前端配置文件：`vite.config.ts`

```typescript
export default defineConfig({
  server: {
    port: 3000,           // 前端端口
    proxy: {
      '/api': {
        target: 'http://localhost:8080',  // 后端地址
        changeOrigin: true,
      },
    },
  },
})
```

## 🚨 常见问题与解决方案

### 1. 端口冲突问题

**问题**: `Port 3000 is in use, trying another one...`

**解决方案**:
```bash
# 查找占用端口的进程
netstat -ano | findstr :3000
netstat -ano | findstr :8080

# 结束占用进程
taskkill /PID <进程ID> /F

# 或者修改配置文件中的端口
# 前端：vite.config.ts 中的 port 配置
# 后端：application.yml 中添加 server.port 配置
```

### 2. 数据库连接失败

**问题**: `java.sql.SQLException: Access denied for user 'root'@'localhost'`

**解决方案**:
1. 检查MySQL服务是否启动
2. 验证数据库用户名密码
3. 确认数据库名称正确
4. 检查防火墙设置

```bash
# 测试数据库连接
mysql -u root -p
USE bookstore;
SELECT 1;
```

### 3. 依赖安装失败

**问题**: `npm install` 失败

**解决方案**:
```bash
# 清除npm缓存
npm cache clean --force

# 删除node_modules和package-lock.json
rm -rf node_modules package-lock.json

# 重新安装
npm install

# 使用淘宝镜像源
npm config set registry https://registry.npmmirror.com
```

### 4. Maven编译失败

**问题**: `mvn compile` 失败

**解决方案**:
```bash
# 清理Maven缓存
mvn clean

# 跳过测试编译
mvn compile -DskipTests

# 检查Java版本
java -version
mvn -version
```

### 5. 前端页面空白

**问题**: 页面显示空白，控制台报错

**解决方案**:
1. 检查后端服务是否启动
2. 确认API代理配置正确
3. 查看浏览器控制台错误信息
4. 检查网络连接

### 6. 跨域问题

**问题**: 前端无法访问后端API

**解决方案**:
```yaml
# 在后端application.yml中添加CORS配置
spring:
  web:
    cors:
      allowed-origins: "http://localhost:3001"
      allowed-methods: "GET,POST,PUT,DELETE,OPTIONS"
      allowed-headers: "*"
      allow-credentials: true
```
